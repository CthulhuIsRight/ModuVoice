# ModuVoice

ModuVoice is a universal, modular TTS voice generator inspired by DAW plugin systems. Swap between engines (espeak, Mimic, etc.) and layer effects like equalization and oscillation to craft unique voice personas. Ideal for games, interactive media, and creative projects. Open-source and runs offline.
